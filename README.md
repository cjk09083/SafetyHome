## 담당
- Android(JAVA) 프론트/백엔드 개발
- WebRTC (Kakao I connect) 영상통화 및 녹화 기능
- Firebase Cloud Messaging (FCM) 활용
- FFmpeg 영상 및 음성 편집&합성
- GPS 위치 공유
